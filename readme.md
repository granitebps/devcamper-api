# DevCamper API

> Backend API for DevCamper application, which is bootcamp directory website

## Usage

Rename "config/config.env.env" to "config/config.env" and update the values/settings to your own

## Install Dependencies

```bash
npm install
```

## Run app

```bash
# Run in dev mode
npm run dev

# Run in prod mode
npm start
```
